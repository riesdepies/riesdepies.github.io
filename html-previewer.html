This utility page renders HTML, CSS, and JavaScript code snippets securely from a URL hash, isolating them from domain restrictions and strict Content Security Policies.

How to use:

  2. Automated tools (such as browser extensions or userscripts) can dynamically feed code blocks directly to this page.
  3. The page will automatically parse and execute the script in a clean execution context.
