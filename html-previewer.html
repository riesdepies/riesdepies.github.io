Gemini HTML Previewer

This is a utility page designed to render HTML code blocks generated by Google Gemini securely, isolated from domain restrictions and strict Content Security Policies.

How to use this page:
  1. Ensure you have the custom Tampermonkey script active on gemini.google.com.
  2. Ask Gemini to write an HTML page, component, or snippet.
  3. Click the Play button added to the top right of the code block.
  4. The script will encode the code into the URL hash and automatically open this page to execute it.
